# Telegram Бот на Python

## 🚀 Как запустить

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Вставь токен, который ты получил от @BotFather, в файл `bot.py` вместо `ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН`.

3. Запусти бота:
```bash
python bot.py
```

- `/start` — приветствие
